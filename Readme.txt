很高兴你能使用我的小软件：色度学计算工具（基于传统指标）。下面对本软件进行详细说明！
      1、本软件的使用方法非常简单，将LSC、TPV样品的透射可见光谱或者led光源的可见光光谱（380-700nm）填入excel中，程序会自动计算出色温、显色指数、色度坐标（1931、1960、1976都有，欢迎任意挑选）！程序完全由python编写，并且打包成exe文件，直接使用即可。

     2、目录里还有国家标准的光源显色性评价方法的国家标准，里面是显色指数的详细计算流程，讲的十分通俗易懂。

     3、精确度和商业软件精度一致或者更高，这得益于我们采用的光谱的三次样条插值法，将光谱拉到了1nm精确度。我还将亚洲女性的肤色R15放入显色指数计算中，以更符合我国评价标准使用。

      4、Duv指标我放在了传统指标里以减少高等色度学TM-30的空间复杂度，如果只是传统指标不需要使用这个。

      5、Yang Chenchen的人眼视见函数可见光透射率AVT计算也一并编写在内。由于使用1nm光谱三次样条插值的原因，所以我的软件计算标准精度会比他这篇论文更高。https://www.sciencedirect.com/science/article/pii/S2542435119302740
